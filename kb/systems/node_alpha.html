● Alpha Substrate Planet Secondary processing layer. Where Alpha Core handles active computation, Alpha Substrate stores the Collective's backup instances — minds archived in crystalline matrices beneath the surface. The population is technically infinite.
⬡ Node Alpha Processing Station Station The Collective's primary backup node. If Nexus Prime goes dark, Node Alpha assumes primary network coordination within three ticks. The station processes more data per cycle than the entire Solarian research network combined, and it's considered a secondary facility. Organic visitors are accommodated. The station's efficiency makes most of them uncomfortable.
